# Job Search Capstone

A tool to cut down the time spent finding and vetting job listings so it can go toward networking and skills growth instead.

## Problem

Job searching eats hours every weekday:
- Too many listings to sift through manually.
- Most listings are stale or no longer active.
- Applications routed through ATS systems often never reach a human reviewer.
- Serious consideration usually requires applying the same day a role is posted.

## What it does (v1)

Given a set of job titles/keywords and a location or remote preference, the tool:
1. Finds matching job listings on LinkedIn.
2. Validates each listing by checking for a matching posting on the hiring company's own career page.
3. Opens the verified, active listing on the company's official career site for manual submission.

Manual submission is intentional for v1 — this tool finds and verifies, it doesn't auto-apply.

## Roadmap (v2+)

- Tailor a resume to the specific role.
- Draft a cover letter for the role.
- Generate editable .DOCX files for both.

## Stack

HTML/CSS/JS, Supabase, Vercel, GitHub.

## Status

Early development — built first as a personal tool for my own job search, with an eye toward opening it up to other job seekers later.
